**MAmotif** is an integrative toolkit for searching cell type-specific co-factors associated with differential binding.

Features
--------

- Quantitatively compare ChIP-Seq samples by robust linear regression on common protein binding sites(peaks) (MAnorm model)
- Scan for motif target sites and perform motif enrichment analysis on genomic regions
- Identify transcriptional factors (TFs) associated with the cell-type differential binding
- Search for candidate cell-type specific regulators
